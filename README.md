mkvmerge-static-linux
=====================

> A static binary for the [`mkvmerge(1)`][mkvmerge] command for Linux.

## Installation

```sh
$ npm install mkvmerge-static-linux
```

## Usage

```sh
const mkvmergeStatic = require('mkvmerge-static-linux')
console.log(mkvmergeStatic.path)
```
